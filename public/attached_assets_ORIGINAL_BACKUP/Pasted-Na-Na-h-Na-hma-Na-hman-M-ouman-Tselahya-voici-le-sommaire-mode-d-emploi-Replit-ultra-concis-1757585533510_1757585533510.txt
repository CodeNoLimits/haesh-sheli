# Instructions Replit (pas à pas)

1. **Créer le projet**

   * `npx create-next-app@latest haesh-sheli --typescript --tailwind --app`
   * Ouvrir sur **Replit** (ou importer repo si déjà créé).
2. **Dépendances clés**

   * UI/UX : `npm i framer-motion lucide-react @tailwindcss/typography @tailwindcss/forms @tailwindcss/aspect-ratio`
   * Données : `npm i prisma @prisma/client`
   * Paiement/Email (si choisis) : `npm i stripe @sendgrid/mail` (ou PayPal SDK)
3. **Base de données**

   * `npx prisma init` → config `DATABASE_URL` (Postgres géré, Railway/Supabase au besoin)
   * Coller le **schema Prisma** de l’audit (Product/Category/User/Order).
   * `npm run db:push` (ou `prisma migrate dev`) puis `npm run db:seed` si prévu.
4. **Design system**

   * Config **Tailwind** (thème hybride rouge Breslev + accents modernes).
   * Créer composants : `Header`, `Footer`, `Navigation`, `ProductCard`, `CartDrawer`, `CheckoutForm`, `SearchBar`.
5. **Catalogue & API**

   * Routes `app/api/products` et `app/api/checkout` (depuis le guide).
   * Page `app/(shop)/catalogue` + détails `[slug]` avec variantes (SKU) et stocks.
6. **Multimédia**

   * Composant `VideoSection` + page **“Breslev Live / מדיה”**.
   * Intégrer 3 vidéos Na Nach (YouTube embed), lazy-load, vignettes.
7. **Checkout & trust**

   * Stripe/PayPal (mode test), calcul TVA/port, e-mail confirmation.
   * Badges “Prix du Keren”, “Mission”, politique retours/livraison.
8. **PWA & perf**

   * Config `next-pwa`, `manifest.json`, images optimisées (`next/image`), caches.
9. **i18n & RTL**

   * `i18n` (he par défaut), vérifier **RTL** sur tous les composants, traductions clés (HE/FR/EN).
10. **Analytics & SEO**

* GA4/Hotjar, `next-seo`, `Schema.org Product`, sitemap/robots, méta OpenGraph.
